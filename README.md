# AI Chatbot Starter

A clean starter template for a React + Vite + Tailwind frontend and a FastAPI backend.

## Project structure

- frontend/ — React + Vite + Tailwind UI
- backend/ — FastAPI API server

## Frontend

1. cd frontend
2. npm install
3. npm run dev
4. npm run build

## Backend

1. cd backend
2. python -m venv .venv
3. source .venv/bin/activate   # Windows: .venv\Scripts\activate
4. pip install -r requirements.txt
5. uvicorn main:app --reload
