# AI-Powered Knowledge Management and Query Resolution System - Frontend

## Overview

This repository contains the frontend application for the AI-Powered Knowledge Management and Query Resolution System.

The system enables employees to:

* Search organizational knowledge
* Ask questions through an AI-powered assistant
* Access policies, manuals, and documentation
* View AI-generated responses
* Automatically generate support tickets for unresolved queries

Administrators can:

* Manage support tickets
* Maintain the knowledge base
* Monitor analytics and system performance
* Configure system settings

### User Portal
